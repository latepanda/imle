Getting started

Page setup

Use the mc-theme classes on the body, then compose pages from the blocks shown below.

The library keeps page background, focus states, responsive spacing, and motion preferences consistent across the site.

<link href="assets/styles/mercury.css" rel="stylesheet">

<body class="mc-theme mc-theme--mercury">
  <a class="mc-skip-link" href="#main-content">Skip to content</a>
  <main id="main-content" class="mc-container">
    <section class="mc-hero">...</section>
    <section class="mc-section-card">...</section>
  </main>
</body>

System light mode

Mercury uses @media (prefers-color-scheme: light) to remap the core colour variables between light and dark mode depending on the users system preference.

Use the theme toggle in this guide to preview the same variable remap without changing your system settings. 
@media (prefers-color-scheme: light) {
  :root {
    --mc-color-bg: #efe7f4;
    --mc-color-accent: #17121c;
    --mc-color-muted: #5c5067;
    --mc-color-paper: #ffffff;
  }
}
